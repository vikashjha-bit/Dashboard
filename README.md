# Team Dashboard (React)

This is a frontend dashboard application built using React and Framer Motion.  
The main focus of this project is to create a clean and interactive dashboard UI with smooth animations.

## Features

- Responsive dashboard layout
- Smooth UI animations using Framer Motion
- Loading state for better user experience
- Simple and reusable component structure
- Static data (can be connected to APIs later)

## Tech Stack

- React (Create React App)
- Framer Motion
- CSS

## Installation & Setup

1. Clone the repository or extract the zip file
2. Open the terminal inside the project folder
3. Install the dependencies:
   ```bash
   npm install
4. npm start
